# **App Name**: E&F WorkBee

## Core Features:

- User Registration & Role Selection: Allows users to register and select their role (Worker/Job Seeker, Customer/Employer, Admin).
- Profile Creation & Verification: Enables users to create profiles with required information and undergo verification processes before becoming active.
- Admin Approval System: Implements an admin approval process for workers and employers to ensure trust and safety.
- Location-Based Worker Search: Allows customers to search for approved workers/service providers within a specific location.
- Job Posting & Matching: Enables employers to post jobs/orders, which are then matched with approved workers based on skills and location.
- In-App Communication & Payment: Facilitates secure in-app communication and payment processing for all transactions.

## Style Guidelines:

- Primary color: Navy blue (#1A237E) to convey trust, security, and professionalism.
- Background color: Light gray (#EEEEEE) to create a clean and neutral backdrop.
- Accent color: Teal (#008080) to highlight important actions and information.
- Body and headline font: 'Inter', a sans-serif font providing a modern, machined, objective, neutral look.
- Use clear and professional icons to represent different job categories and services.
- Maintain a clean and organized layout with a focus on easy navigation and information accessibility.
- Use subtle and professional animations to enhance user experience without being distracting.
